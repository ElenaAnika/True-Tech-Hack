## Запуск проекта

1. Склонировать репозиторий и перейти в папку проекта
```bash
git clone https://github.com/zeroqs/True-Tech-Hack
cd True-Tech-Hack
```

2. Установить зависимости
```bash
npm install
```

3. Запустить проект
```bash
npm run dev
```
